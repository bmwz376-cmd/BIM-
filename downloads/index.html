    📋 利用上の注意¶

    ライセンス¶

    本教材は 学習目的での使用 を想定しています。

    推奨される使用方法¶

    ✅ OK - 個人学習での使用 - 試験対策での利用 - 印刷して手元に置く - 社内勉強会での参照

    ❌ NG - 商用利用 - PDFの再配布 - 内容の無断転載 - 改変しての配布

    最新版の入手¶

    最新版は常にこのWebサイトで公開されます。
    📅 定期的にご確認ください。

    🛠️ 自動生成について¶

    これらのファイルは MASTER.md を元に自動生成されています。

    再生成方法¶

    cd /home/user/webapp/bim-textbook-series
python src/build.py

    以下のファイルが自動生成されます： - PDF（VOL1、VOL2） - 図表（PNG × 15種類） - 章ファイル（Markdown × 15ファイル）
