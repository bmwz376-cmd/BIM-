    BIM Certification Textbook Series¶

    ğŸ“˜ Overview¶

    Complete Textbook for BIM Certification (Level 2 & Semi-Level 1)

    This textbook series aims to help you understand BIM not just as "3D CAD software operation," but as "information design throughout the entire building production process."

    ğŸ¯ Target Certifications¶

    VOL1: BIM Certification Level 2¶

    • Exam Format: Multiple Choice (Mark Sheet)
    • Exam Duration: 90 minutes
    • Passing Score: 60+ points out of 100
    • Main Topics:
    • BIM Definition and Concepts (20%)
    • Building Production Process (15%)
    • BIM Model Structure (20%)
    • Standards like LOD & IFC (20%)
    • BIM Implementation Benefits (15%)
    • Terminology & Standards (10%)

    VOL2: BIM Certification Semi-Level 1¶

    • Exam Format: Practical Exam (BIM Software Operation)
    • Exam Duration: 120 minutes
    • Evaluation Criteria: Modeling accuracy, information precision, downstream considerations
    • Main Evaluation Points:
    • Level & Grid accuracy
    • Proper element placement and constraints
    • Modeling with downstream processes in mind
    • Information completeness and consistency

    âŒ Common Misconceptions¶

    Misconception Correct Understanding
    BIM = 3D modeling BIM = Integrated building information management process
    BIM = Software name (Revit, etc.) BIM = Concept; Revit is one implementation tool
    LOD = 3D precision LOD = Information certainty & reliability
    BIM implementation = Instant efficiency Initial learning cost, mid-to-long term benefits
    IFC conversion is perfect Information loss can occur (operational design is key)
