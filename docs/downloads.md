## 🛠️ 自動生成について

これらのファイルは `MASTER.md` を元に自動生成されています。

### 再生成方法

```bash
cd /home/user/webapp/bim-textbook-series
python src/build.py
```

以下のファイルが自動生成されます：
- PDF（VOL1、VOL2）
- 図表（PNG × 15種類）
- 章ファイル（Markdown × 15ファイル）
