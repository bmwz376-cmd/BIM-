# VOL2: 準1級対応（実技編）

## 📗 概要

**BIM利用技術者試験 準1級**に対応した実技試験対策教材です。実際にBIMソフトを操作し、正確なモデリング技術を習得します。

---

## 🎯 試験情報

- **試験形式**: 実技試験（BIMソフト操作）
- **試験時間**: 120分
- **使用ソフト**: Autodesk Revit（バージョンは試験案内参照）
- **評価基準**: モデリング精度、情報の正確性、後工程への配慮

## 📊 学習の進め方

### ステップ1: 基本操作の習得（第1～2章）
Revitの基本操作を理解し、「後工程を考える」という思考法を身につけます。

### ステップ2: 減点回避（第3～4章）
採点基準を理解し、よくある失敗パターンを徹底的に学びます。

### ステップ3: 試験対策（第5章）
試験環境での効率的な作業手順を習得します。

### ステップ4: 実践演習（第6章）
模擬課題で実力を確認。チェックリストで自己採点を繰り返します。

## ⚠️ 合格のための重要ポイント

### 1. レベル設定の徹底
- 壁の下端は**下階レベル**
- 壁の上端は**上階レベル**（数値入力NG）
- 床は**そのレベル**

### 2. グリッドの活用
- 柱は**グリッド交点**
- 壁は**グリッドに沿う**
- グリッド名は**計画通り**

### 3. 制約の理解
- 寸法制約を活用
- EQ（等間隔）を使う
- レベル参照を徹底

### 4. 後工程への配慮
- 部材名をわかりやすく
- 適切な分割
- 情報の付与

### 5. 時間管理
- 120分を有効活用
- チェック時間を確保
- 焦らず確実に

## 💡 学習のヒント

1. **手を動かすことが最重要** - 読むだけでなく、実際にRevitで操作しましょう
2. **失敗から学ぶ** - 第4章の失敗例を必ず自分で再現してください
3. **チェックリストを活用** - 毎回の演習でチェックリストを使いましょう
4. **時間を計る** - 本番同様の時間制限で練習しましょう
5. **繰り返し練習** - 模擬課題を最低3回は繰り返しましょう
