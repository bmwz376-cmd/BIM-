# BIM利用技術者試験 教科書シリーズ

## 📘 教材概要

**BIM利用技術者試験（2級・準1級）対応の完全教科書**

この教材シリーズは、BIMを単なる「3D CADソフトの操作」ではなく、**「建築生産プロセス全体における情報設計」**として正しく理解することを目的としています。

---

## 🎯 対象試験

### VOL1: 建設業経理士2級対応
- **試験形式**: 選択式（マークシート）
- **試験時間**: 90分
- **合格基準**: 60点以上/100点満点
- **主な出題分野**:
  - BIMの定義と概念 (20%)
  - 建築生産プロセス (15%)
  - BIMモデルの構造 (20%)
  - LOD・IFC等の標準 (20%)
  - BIM導入効果 (15%)
  - 用語・規格 (10%)

### VOL2: 建設業経理士準1級対応
- **試験形式**: 実技試験（BIMソフト操作）
- **試験時間**: 120分
- **評価基準**: モデリング精度、情報の正確性、後工程への配慮
- **主な評価項目**:
  - レベル・グリッドの正確性
  - 部材の適切な配置と制約設定
  - 後工程を考慮したモデリング
  - 情報の完全性と整合性

## 🔑 重要な学習ポイント

### ❌ よくある誤解

| 誤解 | 正しい理解 |
|---|---|
| BIM = 3Dモデル作成 | BIM = **建築情報の統合管理プロセス** |
| BIM = Revit等のソフト名 | BIM = 概念、Revitは実装ツールの一つ |
| LOD = 3Dの精密度 | LOD = **情報の確定度・信頼性** |
| BIM導入で即効率化 | 初期は学習コスト、中長期で効果 |
| IFC変換は完璧 | 情報ロスが起こり得る（運用設計が重要） |

### ✅ 合格のための3つの柱

1. **BIMの本質理解** - 3D操作ではなく情報設計として捉える
2. **標準・規格の理解** - LOD、IFC、OPEN BIMの意味を正確に
3. **実務的な運用知識** - 後工程を考えたモデリング思考
