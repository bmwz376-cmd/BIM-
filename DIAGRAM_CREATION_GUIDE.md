```

アクセントグリーン: #4CAF50 （成功、OK、完了）
警告オレンジ: #FF9800 （注意、重要ポイント）
エラーレッド: #F44336 （NG、エラー、問題）
セカンダリブルー: #5B9BD5 （補助要素）
ライトグレー: #F5F7FA （背景）
ダークグレー: #2C3E50 （テキスト）
ボーダーグレー: #E0E0E0 （枠線）
ホワイト: #FFFFFF （背景、カード）
```

### フォント指定
```
タイトル: 游ゴシック Bold 28pt
見出し: 游ゴシック Bold 24pt
サブ見出し: 游ゴシック Medium 18-20pt
本文: メイリオ Regular 14-16pt
注釈/補足: メイリオ Regular 12pt
キャプション: メイリオ Light 11pt
```

### 共通仕様
- **画像サイズ**: 1600×1000px（16:10比率）
- **解像度**: 150 DPI以上
- **ファイル形式**: PNG（透過なし）
- **背景色**: #FFFFFF（白）
- **余白**: 上下左右 80px以上
- **図形の角丸**: 8-12px
- **影**: rgba(0,0,0,0.1) offset 4px blur 12px
